Sistema de Control de Asistencia para Servicio Social

Edgar Calderón Bahena

Aplicación móvil desarrollada en Android con Kotlin, diseñada para automatizar y digitalizar el registro de asistencia de alumnos que realizan su servicio social. Permite a los estudiantes registrar su entrada y salida mediante código QR o contraseña, llevando un control automático de las horas acumuladas. Cuenta con un módulo administrativo que permite visualizar, gestionar y modificar la información de cada alumno, así como consultar su historial de asistencias y progreso hacia las 420 horas requeridas. La información se almacena en una base de datos MySQL gestionada a través de una API REST desarrollada en PHP.
